Instructions

Try to escape the hedge maze without getting killed by a gnome.

You have an assortment of items to help you escape, but you only get one of each so use them wisely.

Shovel: Allows you to dig up one hedge in front of you.

Ladder: Once placed, you can use it to climb on top of hedges.

Sword: Helps you kill annoying gnomes.

Jump Boost: Allows you to jump across a gap when on top of a hedge.

Controls

Arrow Keys: Move your character

Number Keys 1-4: Select tool

Spacebar: Use selected tool

Q: Quit to menu

About

This game was created by Ryan Breuer for the Ludum Dare 28 competition. The theme of the competition was 'You only get one'.
